## Objetivo

O Agent Skills Manager é uma extensão VS Code para gerenciar skills e rules de agentes de IA a partir de um repositório fonte.

- [Arquitetura e Guia de Desenvolvimento](architecture-and-dev-guide) — stack técnica, estrutura do monorepo, arquitetura do sistema, setup de ambiente, comandos, testes e erros comuns.
- [Referência de API e Roadmap](api-reference-and-roadmap) — configuração, comandos, formatos de skills/rules, destinos de sync, TreeView, roadmap de implementação e glossário.
