# Agent Skills Manager

> VS Code extension for managing, syncing, and validating AI agent skills and rules from a centralized git repository.

## Overview

Agent Skills Manager is a VS Code extension that lets you:

- **Discover** skills and rules in a configured source directory (e.g., `.agents/`)
- **Validate** frontmatter and structure of skill/rule files
- **Synchronize** content to configured destinations (Claude, GitHub Copilot, custom paths)
- **Activate/deactivate** items per workspace via an interactive TreeView
- **Watch** for external changes and auto-sync when enabled
